How does it Work

  1. Click "Apply for Geohazard Certification"
  2. Choose project type.
  3. Input proponent's details.
  4. Upload signed letter request addressed to the Regional Director.
  5. Click "Submit and proceed to project details."
  6. Input project details and upload the corresponding photos per project.
  7. Review details then submit.
  8. Keep the reference number to track the status of your request.
